# installation

> [!IMPORTANT]
> Please see [Dependecies](#Dependecies)
 
```bash
git clone  https://github.com/No0ne003/dotfiles
cd dotfiles
# manually
cp -r .config/* ~/.config
cp .zshrc ~/.
# using Stow
stow .
```
